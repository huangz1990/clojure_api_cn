compile
--------

**(compile lib)**

编译符号(symbol) ``lib`` 定义的命名空间到一系列的class文件。 ``lib`` 的源文件必须在正确的类关联(classpath-relative)的目录下，输出的文class文件将会在 ``*compile-path*`` 定义的目录里，这个目录也必须在类路径(classpath)里面。
